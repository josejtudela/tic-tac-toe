# Tic-Tac-Toe

Tic-tac-toe (also known as noughts and crosses or Xs and Os) is a game for two players, React and Vue, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

You have two panel, "Game" and "Statistics", in the first you can play Tic-tac-toe game and the second panel you can see your matches statistics.

Enjoy it!

## Install

```shell
git clone https://github.com/josejtudela/tic-tac-toe.git
cd tic-tac-toe
npm install
cd client
npm install
```

## Start

```shell
cd tic-tac-toe
npm start
npm run client
```

## Test

```shell
npm test
```
